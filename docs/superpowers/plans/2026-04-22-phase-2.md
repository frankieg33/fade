# Phase 2 Implementation Plan

**Goal:** Ship rename, icon picker (groups + apps), move-app menu, restore-defaults, and fix the search-filter bug.

**Architecture:** Additive serde fields on `Bucket` / `AppRule`; new `icon_catalog.rs` module; new Slint state properties and callbacks; Rust helpers for icon resolution and customized-detection.

**Tech Stack:** Rust, Slint, TOML/serde, Nerd Font MDI glyphs.

---

## Task 1: Data model — icon fields + helpers

**Files:**
- Modify: `src/config.rs`
- Modify: `src/main.rs` (icon_for_* call sites in build_groups / build_unassigned_rules)
- Modify: `src/buckets.rs`, `src/monitor.rs` if Bucket/AppRule literals exist (compiler will catch)

Steps:

- Add `#[serde(default)] pub icon: Option<String>` to `Bucket` and `AppRule`.
- Add `Config::icon_for_bucket(idx) -> String` returning `bucket.icon.clone().unwrap_or_else(|| icons::bucket_icon(&b.name).to_string())`.
- Add `Config::icon_for_app(process) -> String` returning `app_rule.icon` if present else `icons::process_icon(process)`.
- Add free fn `fn app_is_customized(bucket: &Bucket, rule: &AppRule) -> bool` returning `rule.timeout_mins != bucket.timeout_mins || rule.action != bucket.action`.
- Tests:
  - `test_icon_fields_default_to_none_in_old_toml`
  - `test_icon_fields_roundtrip`
  - `test_app_is_customized_detects_timeout_diff`
  - `test_app_is_customized_detects_action_diff`
  - `test_app_is_customized_ignores_icon_only_diff`
  - `test_icon_for_app_falls_back_to_catalog`
  - `test_icon_for_app_uses_override_when_set`
- Update `build_groups` and `build_unassigned_rules` in `main.rs` to use the helpers and the new `app_is_customized`.
- Run `cargo test`.
- Commit: `feat: add icon override fields to Bucket and AppRule`

## Task 4: Icon picker UI

**Files:**
- Modify: `ui/main.slint`
- Modify: `src/main.rs`

Steps:

- Define new struct `IconEntry { glyph: string, keywords: string }` in Slint.
- Properties:
  - `in-out property <int> icon-picker-group: -1;` (group being targeted; -1 = closed)
  - `in-out property <int> icon-picker-app: -1;`  (app idx within that group; -1 = picking group icon, not app)
  - `in-out property <string> icon-search-text: "";`
  - `in property <[IconEntry]> icon-results;`
- Callbacks: `icon-search-changed(string)`, `set-group-icon(int, string)`, `set-app-icon(int, int, string)`.
- UI: floating overlay (like drawer) shown when `icon-picker-group != -1`. Search LineEdit at top wired to `icon-search-text` with `changed text => root.icon-search-changed(self.text)`. ScrollView with a grid-ish layout (HorizontalBox wrapping rows) of Rectangles showing glyph; click → emit `set-group-icon` or `set-app-icon`, close picker.
- Rust: on `icon_search_changed`, run `icon_catalog::search(query)`, populate `icon_results` VecModel. On `set_group_icon`, write `bucket[idx].icon = Some(glyph)`, save, refresh. On `set_app_icon`, find-or-create AppRule with bucket's current timeout/action and set its `icon`.
- Wire the group header icon Text and app-row icon Text so clicking opens the picker. Use a TouchArea.
- Test manually.
- Commit: `feat: icon picker with searchable glyph grid`

## Task 5: Move app between groups

**Files:**
- Modify: `ui/main.slint`
- Modify: `src/main.rs`

Steps:

- Properties: `in-out property <int> kebab-group: -1;` `in-out property <int> kebab-app: -1;`
- Callbacks: `move-app(int /*from*/, int /*app*/, int /*to*/)`, `remove-from-group(int, int)`.
- UI: Add kebab button (`\u{F01D9}`) to each app row, right of Edit/Reset. Click → sets `kebab-group = g-idx; kebab-app = a-idx;`.
- Popup: floating menu anchored near the kebab when indices match. Lists all group names with g-idx != from-group (using `for g[i] in root.groups`), plus a separator and "Remove from group". Click group → `move-app`; click remove → `remove-from-group`; click outside → close (invisible background TouchArea covering rest of window).
- Rust: `on_move_app`: remove process from `bucket[from].processes`, push to `bucket[to].processes`, save, refresh. `on_remove_from_group`: remove process from bucket — AppRule stays and surfaces in the unassigned section.
- Test manually.
- Commit: `feat: move or unassign apps via kebab menu`

## Task 7: Search filter (bug fix)

**Files:**
- Modify: `ui/main.slint`
- Modify: `src/main.rs`

Steps:

- Slint: add `callback search-changed(string)` to fire when search-text changes; wire `changed text => root.search-changed(self.text)` on the top LineEdit.

- Test manually: typing filters groups + apps, clearing restores all.
- Commit: `fix: wire search bar to filter groups and apps`
